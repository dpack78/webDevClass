Unix shell command cheat sheet

David Packer HW 2

Yet another Unix shell cheat sheet as the internet does not have enough of these already.

Command Name Description Example
ls List current Directory ls
cd Change Working Directory cd documents
mkdir Make new directory mkdir myNewFolderName
less View a file with page scrolling less myFileIWantToView
vim open a file with vim vim myFile
rm removes a file rm myBadFile
shutdown turns off the computer shutdown
reboot restart the computer reboot
date Show the current date and time date
grep Look for a character pattern in files grep 1234 myFileToBeSearched
